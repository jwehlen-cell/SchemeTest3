üìù Sample Queries

List All Stations

Get basic information about all seismic stations

SELECT sta, lat, lon, elev, staname FROM site ORDER BY sta LIMIT 20;

Recent Earthquakes

Find recent earthquake origins with magnitudes

SELECT orid, lat, lon, depth, time, mb, ms FROM origin WHERE time > extract(epoch from now() - interval '7 days') ORDER BY time DESC;

Station Networks

Show station network affiliations

SELECT s.sta, s.staname, a.net, n.netname FROM site s JOIN affiliation a ON s.sta = a.sta JOIN network n ON a.net = n.net ORDER BY a.net, s.sta;

Magnitude Statistics

Calculate magnitude distribution

SELECT ROUND(magnitude, 1) as mag_rounded, COUNT(*) as count FROM netmag WHERE magnitude IS NOT NULL GROUP BY ROUND(magnitude, 1) ORDER BY mag_rounded;
